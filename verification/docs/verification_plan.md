# AXI Crossbar 验证计划（测试点分解）

## 1. 验证目标

验证 4×4 AXI Crossbar 的功能正确性：
- 地址路由正确性
- 数据传输完整性
- AXI4 协议合规性
- 并发处理能力
- 错误处理能力
- 边界条件处理
- 性能指标
